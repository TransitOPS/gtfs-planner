# Prototype Research Files

The HTML and CSS files in this directory are fixture-only research artifacts. They are not a
supported product surface and are not governed by current Phoenix/LiveView or design-system
contracts.

These files remain packaged under the application `priv/` directory in OTP releases for local
inspection, but they have no production route, controller, navigation link, or static-file
publication path. They must not be routed, linked, or added to `GtfsPlannerWeb.static_paths/0`
without a new product decision.
